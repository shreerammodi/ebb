# ebb

**ebb** is a local-first, privacy-centric, keyboard-first web app for flowing
competitive debate rounds. All data lives in your browser; there is no backend.

## Getting started

To start a local web instance, run:

```bash
npm install
npm run dev
```

To generate an installer for the desktop app (preferred), run:

```bash
npm run desktop:build
```

Or you could run a live server for the desktop app using

```bash
npm run desktop:dev
```
